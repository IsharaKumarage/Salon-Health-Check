#Salon Health Check

A deploy-ready React + Vite interactive customer engagement tool for Beautech.

## Features
- Dusty pink highlights and charcoal text
- Interactive salon owner problem scenarios
- Revenue leak calculator
- Salon health result dashboard
- Beautech transformation CTA
- Demo lead form UI
- Responsive layout

## Run Locally
```bash
npm install
npm run dev
```

## Build for Deployment
```bash
npm run build
```

## Deploy on Vercel
1. Upload this folder to GitHub.
2. Open Vercel.
3. Import the GitHub project.
4. Framework preset: Vite.
5. Build command: `npm run build`
6. Output directory: `dist`

## Customize
Main files:
- `src/main.jsx` — app logic and content
- `src/styles.css` — colors, layout, responsive design

Brand colors used:
- Dusty Pink: `#D9828E`
- Charcoal: `#202326`
- Warm Cream: `#F8F1EA`
